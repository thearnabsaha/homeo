## Notes on "এআই" and "রিপার্টরি"

- **এআই** = Bengali script (এ + আ + ই = "e-ā-i" for "AI")
- **রিপার্টরি** = Bengali script for "Repertory"
- **কেন্টের** = Bengali script for "Kent's"

These are Bengali transliterations, not Latin script.
